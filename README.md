# 🚀 INTURNSHIP-Task-4

## 📝 To-Do Web Application

The **To-Do Web Application** is a responsive task management application developed using **HTML, CSS, and JavaScript**. It enables users to efficiently create, organize, and manage their daily tasks through an intuitive and user-friendly interface.

Users can add new tasks, edit existing ones, delete unwanted tasks, assign a **date and time**, and mark tasks as completed. The application helps improve productivity by allowing users to track their progress and manage daily activities in an organized manner.

### ✨ Features

* ➕ Add new tasks with date and time
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 📅 Set due date and time for each task
* 📋 Organize and manage daily activities
* 📱 Responsive design for desktop and mobile devices
* ⚡ Fast and interactive user interface

### 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

### 🎯 Objective

Develop a responsive To-Do Web Application that allows users to add, edit, organize, and delete tasks, assign dates and times, mark tasks as completed, and efficiently manage their daily activities using modern front-end web technologies.
